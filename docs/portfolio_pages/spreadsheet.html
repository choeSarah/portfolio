Microsoft Spreadsheet Clone

Introduction

This project mimics the functionalities of Microsoft's Excel application. Using a model-view-controller architecture, I was able to create a program that accepts variables (cell names), performs basic infix operations, and has file I/O functionality. Furthermore, it is able to split values across cells by commas, semi-colons, or spaces.

Throughout this process, I created tests for quality assurance and coded the operations for variable validity and formula evaluations.
