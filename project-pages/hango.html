HanGo

HanGo is an educational app designed to teach the Korean alphabet and basic vocabulary through giving users writing and vocabulary practice. The app game-ifies the learning experience by adding a typing game for users to practice recalling vocabulary.

In this project, I managed the backend development through agile software development and streamlined task management through Github Projects. I also designed and implemented user interface elements through Figma and QT Creator, as well as code the physics simulations using Box2D to achieve realistic object dynamics.
